# Ambient ANTHROPIC_* env vars that would override ~/.zlaude/settings.json.
# The zlaude wrapper functions strip these (env -u) so the profile's
# settings.json fully controls the model, endpoint, and auth — otherwise a
# project's .env (e.g. pocketshell's ANTHROPIC_MODEL=glm-4.6) leaks in and
# silently overrides the configured GLM-5.1 default.
